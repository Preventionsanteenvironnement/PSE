Module D3 — Thématique D

Les achats

Analyser une situation d'achat.

📋 Situation — L'achat en ligne d'Emma

Emma, 16 ans, commande un téléphone à 350€ sur un site internet. Après réception, elle constate que l'écran a un défaut (pixel mort). Elle veut renvoyer le produit. Par ailleurs, au supermarché, elle hésite entre deux yaourts dont l'un est proche de sa date de péremption.

Exercice 1C1
Indiquer le délai de rétractation pour l'achat en ligne d'Emma.
Explication : Le droit de RÉTRACTATION = 14 JOURS pour tout achat en ligne, SANS justification. Emma peut renvoyer le téléphone et être remboursée dans les 14 jours suivant le retour.
Exercice 2C3
Expliquer la différence entre DLC et DDM pour les yaourts.
Correction : DLC (Date Limite de Consommation) = 'À consommer JUSQU'AU...' → produits PÉRISSABLES (yaourts, viande). Après la DLC = DANGER sanitaire (bactéries). DDM (Date de Durabilité Minimale) = 'À consommer DE PRÉFÉRENCE AVANT...' → produits SECS (biscuits). Après la DDM = perte de QUALITÉ mais pas de danger.
Exercice 3C1
Identifier les informations obligatoires sur l'étiquette d'un yaourt.
Explication : Les mentions OBLIGATOIRES : dénomination, liste des ingrédients, DLC ou DDM, allergènes (en gras), valeurs nutritionnelles, nom du fabricant, origine, conditions de stockage.
Exercice 4C1
Nommer 3 labels de qualité alimentaire.
Explication : LABEL ROUGE = qualité supérieure. AB = sans pesticides chimiques. AOP = savoir-faire local d'une zone géographique précise. Ce sont des signes OFFICIELS de qualité.
Exercice 5C4
Proposer les démarches pour Emma concernant le téléphone défectueux.
Correction : 1) Contacter le vendeur (service client) pour signaler le défaut. 2) Exercer le droit de RÉTRACTATION (14 jours) : renvoyer le produit en gardant une preuve d'envoi. 3) Exiger le REMBOURSEMENT intégral dans les 14 jours. 4) Si le vendeur refuse : contacter une association de consommateurs. 5) Garder TOUTES les preuves (email de commande, photos du défaut, accusé de réception du retour).
