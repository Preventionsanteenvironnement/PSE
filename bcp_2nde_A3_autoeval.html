📖 Rappel : Circuit du Message Moteur

La contraction musculaire est commandée par un message nerveux moteur qui part du cerveau.

Les 4 étapes du circuit :

  1. Cerveau → envoie l'ordre de contraction
  2. Moelle épinière → transmet le message (voie centrale)
  3. Nerf moteur → conduit le message vers le muscle
  4. Muscle → reçoit l'ordre et se contracte

⚡ Réactions du corps à l'effort :

  • ↑ Fréquence cardiaque (pouls)
  • ↑ Fréquence respiratoire
  • ↑ Température corporelle
