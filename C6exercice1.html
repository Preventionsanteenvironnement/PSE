Exercice 2

Connecteurs logiques

6 pts
C6

Consigne

Complète le mini-paragraphe avec les bons connecteurs pour que le raisonnement soit clair.

Choisis le connecteur pour chaque blanc

6 pts
Texte
Je coupe l’alimentation de la machine [1] le capot de protection est mal refermé. Je préviens ensuite le responsable, [2] le risque d’accident augmente si quelqu’un redémarre la machine. [3], on pourra reprendre le travail en sécurité après vérification.
Score : 0 / 6
Astuce : parce que = cause ; donc = conséquence ; cependant = nuance.
