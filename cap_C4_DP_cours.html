📖 Séance 1 — Je repère le danger

Objectif : Nommer un danger au travail.

Durée : 45 min

🧑 Situation : Léa, CAP vente

Léa est en CAP vente. Elle travaille dans un supermarché. Elle porte des cartons très lourds. Le soir, elle a mal au dos.

📄 Document A — Les dangers au travail

  • Porter lourd → mal au dos.
  • Sol mouillé → risque de chute.
  • Couteau → risque de coupure.
  • Bruit fort → mal aux oreilles.
  • Produit ménager → irritation de la peau.

📄 Document B — Dangers selon le métier

MétierDanger principal
VentePorter lourd
RestaurationCoupure, brûlure
CoiffureProduit chimique
MécaniqueBruit, chute
Aide à domicilePorter lourd

📄 Document C — Danger et risque

Un danger est une chose qui peut blesser.

Un risque, c'est la chance que le danger blesse.

Exemple : le couteau est un danger. Se couper est le risque.

💡 Séance 2 — Comment arrive un accident

Objectif : Comprendre les étapes d'un accident.

Durée : 45 min

🧑 Situation : Tom, CAP cuisine

Tom est en CAP cuisine. Il court dans la cuisine. Le sol est mouillé. Il glisse et tombe. Il se fait mal au genou.

📄 Document A — Les 3 étapes d'un accident

1. La personne : Tom.

2. Le danger : sol mouillé.

3. Le dommage : genou blessé.

📄 Document B — Schéma simple

Personne + Danger → Accident → Dommage (blessure).

📄 Document C — Exemples d'accidents

  • Une fille court → sol glissant → elle tombe → bras cassé.
  • Un garçon porte un carton trop lourd → il se fait mal au dos.
  • Une coiffeuse touche un produit → sa main rougit.

C1 Question 5 — Vrai ou Faux

Classe Vrai ou Faux ?

Un accident a toujours une cause.
Le dommage, c'est la blessure.
Le danger est toujours visible.
C3 Question 6

✨ Ce que je retiens — Séance 2

  • Un accident a 3 parties : la personne, le danger, le dommage.
  • Le danger rencontre la personne → accident.
  • Le dommage est la blessure.
  • Je peux agir pour éviter l'accident.

✨ Séance 3 — Les solutions pour se protéger

Objectif : Choisir un équipement de protection.

Durée : 45 min

🧑 Situation : Yanis, CAP mécanique

Yanis est en CAP mécanique. Il travaille sur une voiture. Le bruit des machines est fort. Il ne met pas de casque.

📄 Document A — Les EPI

Les EPI = Équipements de Protection Individuelle.

  • Casque → protège la tête.
  • Gants → protègent les mains.
  • Chaussures de sécurité → protègent les pieds.
  • Lunettes → protègent les yeux.
  • Masque → protège le nez et la bouche.
  • Bouchons d'oreilles → protègent les oreilles.

📄 Document B — Protection collective ou individuelle

Collective (pour tous)Individuelle (pour moi)
BarrièreCasque
Sol antidérapantGants
Aération de l'atelierMasque

📄 Document C — Les 3 règles de Yanis

  1. Je porte mes EPI.
  2. Je respecte les consignes.
  3. Je préviens si je vois un danger.

C1 Question 7 — Vrai ou Faux

Classe Vrai ou Faux ?

Je peux signaler un danger à mon tuteur.

✨ Ce que je retiens — Séance 3

  • Les EPI me protègent.
  • Chaque EPI protège une partie du corps.
  • Collective = pour tout le monde. Individuelle = pour moi.
  • Je dois porter mes EPI au travail.
  • Je signale les dangers.
