Dossier des images pour les exercices "Légender une image".

Comment ajouter une image :
1. Dépose ton fichier image (.jpg, .png, .webp) dans CE dossier.
2. Nom du fichier : court, sans espaces ni accents, en minuscules.
   Exemples OK  : digestion_aliments.jpg, oreille_interne.png
   Exemples KO  : "Schéma Digestion.jpg", "oreille (1).png"
3. Dans l'éditeur d'exercices (éditeur > type "Légender une image"),
   tape le chemin : images_legendes/nom_du_fichier.jpg
