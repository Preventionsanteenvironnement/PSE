Document de référence pour la construction des cours du site PSE.
Volumes horaires officiels : Éduscol — *Proposition de répartition horaire pour les modules de PSE en BAC PRO (oct. 2021)* — total 84 h sur le cycle, 1 h hebdomadaire.

Pour chaque module :
- **Architecture** : volume horaire → nombre de séances de 1 h, chacune avec un objectif spécifique formulé « Être capable de… » (verbe d'action lié à la compétence visée)
- **Banque d'images** : nom de fichier, type (photo réaliste / infographie / schéma / pictogramme), descriptif détaillé, source pressentie (IA générative / officiel à récupérer / manuel scolaire)

Convention de nommage des fichiers : `{module}_s{n}_doc{m}_{slug}.jpg|.png`
Convention de dossier : `PSE/bcp_{niveau}_{module}/images/`

## Conventions de production

- **Photos réalistes [IA]** : tu génères avec Midjourney / DALL-E / Stable Diffusion. Style cohérent : éclairage naturel, scène professionnelle française, pas de texte généré en image.

- **Schémas scientifiques** : style manuel scolaire français (Foucher, Delagrave) — fond blanc, légendes propres, vectoriel.
- **Pictogrammes officiels** : à récupérer sur sites officiels (INRS ED 840, Santé Publique France, ADEME).
- **Documents type [récup.]** : sites institutionnels (santepubliquefrance.fr, inrs.fr, ademe.fr, service-public.fr, geo-risques.gouv.fr).
