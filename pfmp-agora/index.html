Aide

Comment utiliser mon carnet

Un espace de travail réflexif, sérieux, pour prendre du recul sur votre pratique professionnelle.

1. Je me connecte

Cliquez sur « Se connecter » et saisissez votre code élève (anonyme). Votre carnet est alors retrouvé automatiquement à chaque visite, sur le même ordinateur. Vous pouvez aussi Exporter votre carnet en fichier JSON pour le conserver ou le transmettre.

2. Je déclare une PFMP

Dans le tableau de bord, créez une période : année, dates approximatives, type de structure. Jamais de nom ni de lieu.

3. Je travaille librement

Naviguez à votre rythme entre l'accueil, le journal des activités, le savoir-être, les compétences et le bilan. Aucun ordre imposé, aucun blocage : revenez en arrière, complétez plus tard, reliez vos activités aux pôles que vous voulez.

4. Mon travail est enregistré

La sauvegarde est automatique (format JSON). Avec votre code élève, votre carnet est aussi enregistré en ligne : vous le retrouvez sur n'importe quel ordinateur, simplement en vous connectant avec le même code. Sans code, le travail reste sauvegardé sur cet ordinateur uniquement — pensez alors à Exporter votre fichier.
