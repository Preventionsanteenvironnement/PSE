Comment utiliser mon carnet

Un espace de travail réflexif et sérieux pour prendre du recul sur votre pratique professionnelle.

1. Je me connecte

Cliquez sur « Se connecter » et saisissez votre code élève (anonyme). Avec ce code, votre carnet est enregistré en ligne : vous le retrouvez sur n'importe quel ordinateur, simplement en vous reconnectant. Sans code, le travail reste sur cet ordinateur — pensez à Exporter votre fichier.

2. Je déclare une PFMP

Dans le tableau de bord, créez une période : année, dates, durée (en semaines ou en jours), type de structure. Jamais de nom ni de lieu.

3. Je travaille librement

Naviguez à votre rythme entre l'accueil, le journal des activités, le savoir-être, les compétences et le bilan. Aucun ordre imposé, aucun blocage.

4. Mon travail est enregistré

La sauvegarde est automatique (format JSON, en ligne avec votre code). Suppressions toujours confirmées.
