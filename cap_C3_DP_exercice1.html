📖 La situation de Sami

Sami, 17 ans, est apprenti boulanger. Il arrive ce matin dans un nouveau fournil. Son patron lui montre les consignes de sécurité affichées au mur (pictogrammes, règlement, plan d'intervention) et lui demande d'analyser son poste de travail avant de commencer.

Sami doit pétrir la pâte, enfourner les baguettes dans le four à 240°C, nettoyer le pétrin. Le fournil est chaud, bruyant, et l'espace est étroit.

📄 Document A — Instruction ou information ?

Dans une entreprise, on trouve deux types de messages :

  • Une instruction = une conduite à tenir obligatoire. Je dois la respecter.
    Exemple : « Porter les gants de protection ».
  • Une information = un renseignement. Elle me donne une indication.
    Exemple : « Le four est chaud », « Horaire d'ouverture : 6h-19h ».

Supports : fiches de poste, affiches, règlement intérieur, plan d'intervention.

🎨 Document B — Les pictogrammes de sécurité

Il existe 5 familles de pictogrammes :

  • 🚫 Interdiction : rond rouge barré (ex : interdit de fumer).
  • 🟦 Obligation : rond bleu (ex : porter un casque).
  • ⚠️ Avertissement / Danger : triangle jaune (ex : surface chaude).
  • 🟩 Secours / Évacuation : carré vert (ex : sortie de secours).
  • 🔥 Incendie : carré rouge (ex : extincteur).

🔍 Document C — L'outil ITAMAMI

Pour analyser un poste de travail, on observe 4 éléments :

  • Individu : l'âge, la formation, l'expérience, l'état de santé.
  • Tâche : les gestes, les postures, les actions à réaliser.
  • Matériel : les outils, les machines, les équipements.
  • Milieu : le bruit, la chaleur, l'espace, l'éclairage.
